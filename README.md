# Ahzi

A React site for Ahzi's CRM AI readiness work.

## Stack

- React with Vite
- Tailwind CSS v4
- shadcn/ui-style owned components
- Motion for interface animation
- Lucide icons

## Commands

```bash
npm run dev
npm run lint
npm run build
```

## Deployment

The `main` branch deploys to GitHub Pages through the workflow in `.github/workflows/pages.yml`.
